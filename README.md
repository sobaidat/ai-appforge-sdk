# AI App Forge SDK

The **AI App Forge SDK** provides a suite of libraries to interact with the AI App Forge platform, enabling you to integrate powerful AI workflows into your Node.js and React Native applications.

## Packages

- **[@ai-appforge/core](./core)**: Core types and shared utilities.
- **[@ai-appforge/nodejs](./nodejs)**: Client SDK for Node.js applications.
- **[@ai-appforge/react-native](./react-native)**: Client SDK for React Native applications.

## Getting Started

Visit the [Documentation](./docs) for detailed guides and API references.

### Installation

```bash
# For Node.js
npm install @ai-appforge/nodejs

# For React Native
npm install @ai-appforge/react-native
```
